Аудиоредактор форматов wav,mp3.

Автор: Сагалов Леонид

Описание:
Данное приложение предоставляет простейшие возможности аудиоредактора

Состав:
editor_console - содержит классы функций для работы в консольном режиме
main_console - содержит разбор аргументов и метод main запуска консольной версии
editor_GUI - содержит класс функций для работы в графическом режиме
main_GUI - содержит костяк графической оболочки
test - тесты

Консольная версия 

Примеры запуска консольной версии:

- python editor_console --cut new name.wav 2(1/2 name.wav появится в новом файле с именем cut.wav)
- python editor_console --cut this name.wav 2(файл name.wav перезапишется и будет хранить 1/2 часть исходного аудио)

- python editor_console --reverse this name.wav(развернет аудио и запишет в тот же файл)
-python editor_console --reverse new name.wav(развернет аудио и запишет в новый файл)

- python editor_console --speed new name.wav 44100(поменяет частоту дискретизации аудио на 44100, аудио ускоритсяб запишет в новый файл)
- python editor_console --speed this name.wav 8000(поменяет частоту дискретизации аудио на 8000, аудио замедлится, запишет в тот же файл)

- python editor_console --splice name1.wav nsme2.wav(склеит два аудио и запишет в новый файл)

- python editor_console --copy name.wav(создаст копию данного аудио)

Графическая версия

- python main_GUI

при нажатии на кнопки появляется диалоговое окно с выбором файла, после выбора возникает диалоговое окно с вопросом о записи в файл(в функциях резки, разворота, изменения скорости),в функциях копирования и склейки выскакивает диалоговое окно с уведомлением о проделанной работе

кнопка Обзор просто что-бы была,открывает диалоговое окно с выбором файла, но не связана ни с какими действиями.
